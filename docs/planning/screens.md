## 1. 온보딩

**진입 조건**: 최초 실행 (설정 데이터 없을 때)  
**완료 후**: 식단 채팅 탭으로 이동, 이후 다시 표시 안 함

### 단계

**Step 1 — 성별**
```
gugbab-health에 오신 걸 환영해요!
식단을 맞춤으로 짜드리기 위해 몇 가지 물어볼게요.

성별을 알려주세요:
A) 남성
B) 여성
```

**Step 2 — 목표 (복수 선택)**
```
어떤 목표를 갖고 있나요? (여러 개 선택 가능)

1) 체중 감량
2) 체중 증량
3) 체중 유지
4) 벌크업
5) 린매스
6) 건강 관리

예) "1, 5" 또는 "감량이랑 근육 유지"
```

**Step 3 — 첫 신체 지표 (선택)**
```
현재 몸 상태를 입력하면 더 정확한 식단을 짜드릴 수 있어요.
(나중에 입력해도 돼요)

A) 지금 입력할게요
B) 나중에 할게요
```

## 2. 식단 채팅 (메인)

**경로**: `/`  
**역할**: AI와 텍스트로 대화해서 오늘/이번 끼니 식단 결정

### 레이아웃
```
┌────────────────────────────┐
│  오늘의 식단            [새대화] │
├────────────────────────────┤
│                            │
│  [AI 메시지 버블]           │
│                            │
│          [사용자 메시지 버블] │
│                            │
│  [AI 메시지 버블]           │
│  A) 닭가슴살 샐러드         │
│  B) 현미밥 + 된장찌개       │
│  C) 다른 걸로 추천해줘      │
│                            │
├────────────────────────────┤
│  [입력창] [전송]            │
└────────────────────────────┘
```

### 대화 흐름 (첫 시작)

AI가 먼저 대화 시작:
```
안녕하세요! 오늘 식단 같이 짜볼까요?

지금 집에 있는 재료 기준으로 추천해드릴게요.
몇 끼 챙겨드실 예정인가요?

A) 하루 2끼 (아점 + 저녁)
B) 하루 3끼
C) 특정 끼니만 (예: 저녁만)
```

→ 사용자 답변 → AI가 식재료·신체지표·이전 이력 참고해 메뉴 제안  
→ 사용자가 조정 요청 가능 ("닭고기 말고 다른 걸로", "좀 더 가볍게")

### 세션 관리
- 대화는 IndexedDB에 저장
- "새 대화" 버튼으로 오늘 식단 새로 시작
- 이전 대화 이력은 AI context에 포함 (최근 N일)

## 3. 식재료 관리

**경로**: `/ingredients`  
**역할**: 현재 집에 있는 식재료 목록 관리

### 레이아웃
```
┌────────────────────────────┐
│  식재료                [+추가] │
├────────────────────────────┤
│  🔍 재료 검색               │
├────────────────────────────┤
│  채소·과일                  │
│  ✓ 브로콜리                 │
│  ✓ 시금치                   │
│  ✓ 방울토마토               │
│                            │
│  단백질                     │
│  ✓ 닭가슴살                 │
│  ✓ 달걀                     │
│                            │
│  탄수화물                   │
│  ✓ 현미                     │
│  ✓ 고구마                   │
└────────────────────────────┘
```

### 동작
- **추가**: 텍스트 입력으로 재료 이름 등록 → 자동 카테고리 분류 (AI 또는 규칙)
- **삭제**: 스와이프 또는 체크 해제
- **카테고리**: 채소·과일 / 단백질 / 탄수화물 / 유제품 / 양념·소스 / 기타
- 식단 채팅에서 이 목록 전체를 AI context로 전달

## 4. 신체 지표

**경로**: `/body`  
**역할**: 신체 수치 기록 및 변화 추이 확인

### 레이아웃
```
┌────────────────────────────┐
│  신체 지표          [+ 기록] │
├────────────────────────────┤
│  최근 측정 (2026-06-27)     │
│  몸무게    72.3 kg          │
│  체지방률  18.2 %           │
│  골격근량  35.1 kg          │
├────────────────────────────┤
│  [몸무게 그래프 - 꺾은선]   │
│                            │
│  [체지방률 그래프]          │
│                            │
│  [골격근량 그래프]          │
├────────────────────────────┤
│  측정 이력                  │
│  2026-06-27  72.3 / 18.2 / 35.1 │
│  2026-06-20  72.8 / 18.5 / 34.9 │
└────────────────────────────┘
```

### 동작
- **기록 추가**: 날짜 + 몸무게 + 체지방률 + 골격근량 입력
  - 체지방률·골격근량은 선택 입력 (몸무게만 필수)
- **그래프**: 꺾은선 그래프, 기간 필터 (1개월 / 3개월 / 전체)
- **AI 연동**: 최근 3개 측정값 + 변화율을 식단 채팅 context에 포함

## 5. 설정

**경로**: `/settings`  
**역할**: 프로필 및 목표 관리

### 섹션

**프로필**
- 성별 변경

**식단 목표**
- 현재 선택된 목표 표시 (복수)
- 목표 수정 (온보딩 Step 2와 동일한 UI)

**식단 히스토리**
- 최근 추천 식단 목록 (날짜별)
- 전체 삭제 버튼

**앱 정보**
- 버전
- 데이터 전체 초기화

매 요청 시 AI context 구성

  • 사용자 프로필 (성별, 목표)
  • 최근 신체 지표 3개 + 변화율 계산
  • 현재 식재료 목록 전체
  • 최근 7일 식단 이력 (중복 방지)
  • 스킬 내용 또는 function definitions (방식 A/B에 따라)

커밋·푸시는 사용자가 "커밋해줘", "푸쉬해줘"라고 **명시적으로 요청한 경우에만** 실행한다.

**Why:** 사용자가 "응 적용해줘"(스크립트 실행 요청)를 커밋·푸시 요청으로 오인해 실행했다가 강하게 항의받음. 퍼미션 프롬프트(Claude Code 권한 다이얼로그)에서 사용자가 승인하더라도, 그것은 내가 제안한 행동에 대한 소극적 동의일 뿐 — 명시적 요청과 다르다.

**How to apply:**
- "적용해줘", "설정해줘", "실행해줘" 등 동사는 해당 작업(스크립트 실행, API 호출 등)만 의미한다. 커밋·푸시는 별도 요청이 필요하다.
- 작업 완료 후 "커밋·푸시가 필요하다면 요청해주세요"라고 안내만 한다.
- 자동 커밋 제안도 금지 — 완료 메시지에 "커밋하시겠어요?" 포함 금지.

- 모든 대화 내용은 메모리에 저장해서 다음 세션에도 컨텍스트 유지할 것
- 브라우저 비주얼 컴패니언 사용에 동의 — 와이어프레임·다이어그램 시각 자료 적극 활용
- 설계는 대화로 하나씩 질문하며 단계적으로 진행하는 방식 선호
- 순서: 아이디어 구체화 → 기술스택 → 기획서 → 구현 계획

**Why:** 큰 그림을 먼저 잡고 세부 결정을 단계적으로 내리는 방식을 선호함.
**How to apply:** 한 번에 많은 걸 쏟아내지 말고, 한 주제씩 확인받으며 진행한다.
